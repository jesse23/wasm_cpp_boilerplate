# WASM C++ Boilerplate

Boiler plate for WASM C++, created at 20200509

using docker below:
https://hub.docker.com/r/trzeci/emscripten


```bash
# build
npm run build

# test
npm test

# test page -> localhost:8080
npm start

# test nodejs
node ./public/example.js

# test c++ binary
./public/example
```
